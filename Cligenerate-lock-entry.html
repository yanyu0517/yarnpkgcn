yarn generate-lock-entry

生成yarn lock文件。

yarn generate-lock-entry

注意：这个命令并不是为普通开发者所设计的，普通开发者是不需要操作yarn lock文件的。运行这个命令会基于package.json生成yarn lock文件。

$ yarn generate-lock-entry
# THIS IS AN AUTOGENERATED FILE. DO NOT EDIT THIS FILE DIRECTLY.
# yarn lockfile v1
yarnpkg@0.14.0:
  version "0.14.0"
  dependencies:
    babel-plugin-transform-inline-imports-commonjs "^1.0.0"
    babel-runtime "^6.0.0"
    bytes "^2.4.0"
    [...]

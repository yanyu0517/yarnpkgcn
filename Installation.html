安 装

首先需要安装Yarn，下面有多重方式供你选择:
    macOS Windows Linux 其他

选择平台

macOS

如果你没有安装node，那么请安装Node.js

Homebrew

你也可以通过Homebrew package manager安装。

brew update
brew install yarn

脚本安装

对于macOS和大部分Unix环境，通过shell脚本安装是最容易的方法之一。你可以按照下面的步骤安装：

curl -o- -L https://yarnpkg.com/install.sh | bash

tarball手工安装

下载tarball并extract。

cd /opt
wget https://yarnpkg.com/latest.tar.gz
tar zvxf yarn-*.tar.gz
# Yarn is now in /opt/yarn-[version]/

配置环境变量

为Yarn设置PATH环境变量，这样你在任何地方都能够调用Yarn

添加export PATH="$HOME/.yarn/bin:$PATH"到配置文件(比如.profile, .bashrc, .zshrc, 等等。)

Windows

有两种方法在Windows中安装Yarn

下载安装文件

你将会下载一个.msi后缀的Yarn安装文件

在安装前请首先安装 Node.js.

下载安装文件

Chocolatey

Chocolatey包来了

Note:当前不兼容通过Ubuntu在Windows安装Yarn，这个问题正在解决中468

配置环境变量

为Yarn设置PATH环境变量，这样你在任何地方都能够调用Yarn

执行set PATH=%PATH%;C:\.yarn\bin"设置环境变量

Debian/Ubuntu Linux

在Debian或者Ubuntu Linux，你可以通过Debian package repository安装Yarn。首先配置repository:

sudo apt-key adv --keyserver pgp.mit.edu --recv D101F7899D41F3C3
echo "deb http://dl.yarnpkg.com/debian/ stable main" | sudo tee /etc/apt/sources.list.d/yarn.list

在Ubuntu 14.04版本和Debian Stable版本中, 需要配置NodeSource repository安装最新的Node.js（Debian Testing版本Ubuntu 16.04版本已经安装了Node.js，因此可以跳过这步）

安装Yarn

sudo apt-get update && sudo apt-get install yarn

CentOS / Fedora / RHEL

在CentOS、Fedora和RHEL，可以通过我们的RPM安装Yarn

sudo wget https://dl.yarnpkg.com/rpm/yarn.repo -O /etc/yum.repos.d/yarn.repo

如果没有安装Node.js，首先需要配置NodeSource repository：

curl --silent --location https://rpm.nodesource.com/setup_6.x | bash -

安装Yarn：

sudo yum install yarn

Arch Linux

Arch Linux系统通过AUR安装。

如果你使用AUR Helper类似yaourt，那么只需要执行：

yaourt -S yarn

Solus

Solus系统通过Solus repository安装。

sudo eopkg install yarn

设置环境变量

为Yarn设置PATH环境变量，这样你在任何地方都能够调用Yarn

添加export PATH="$HOME/.yarn/bin:$PATH"到配置文件(比如.profile, .bashrc, .zshrc, etc.)

其他

如果你使用的系统不在前面描述的范围内，那么可以尝试按照下面的步骤操作： 如果没有安装Node.js，首先安装Node.js

一般的linux发行版本，诸如debian，ubuntu和centos，推荐通过我们提供的安装包来安装yarn。

npm安装

还可以通过npm安装Yarn。 如果没有安装Node.js，那么首先安装Node.js，npm会随着Node.js一起安装好。

安装好npm后，执行：

npm install --global yarn

执行set PATH=%PATH%;C:\.yarn\bin"置环境变量

测试Yarn是否安装好了 

yarn --version

当你安装好Yarn后，就可以开始Yarn了。下面列出一些常用的命令，会对你有帮助。

新建项目

yarn init

添加依赖

yarn add [package]
yarn add [package]@[version]
yarn add [package]@[tag]

更新依赖

yarn upgrade [package]
yarn upgrade [package]@[version]
yarn upgrade [package]@[tag]

删除依赖

yarn remove [package]

根据package.json文件为项目安装所有依赖

yarn

or

yarn install

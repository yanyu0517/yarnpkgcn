yarn unlink

取消之前为包创建的符号链接

要移除之前通过yarn link命令为包创建的符号链接, 可以使用yarn unlink命令

yarn unlink

在之前创建符号链接的文件夹中运行 yarn unlink

yarn unlink [package]

要在项目开发期间为当前包取消已被创建的符号链接, 只需要运行yarn unlink [package]. 您将需要运行 yarn 或者 yarn install 来重新安装已链接的软件包.

yarn link 的另一个示例: 假设存在同一目录下的两个文件夹 react 和 react-relay，其中 react 链接到了react-relay 工程中:

$ cd react
$ yarn unlink
yarn link vx.x.x
success Unregistered "react".

$ cd ../react-relay
$ yarn unlink react
yarn link vx.x.x
success Unregistered "react".

新建一个项目

对于现有的项目，或者新开始一个项目，Yarn的工作流程都是一样的

在你想添加Yarn的目录（通常都是项目的根目录），执行下面的命令

yarn init

Yarn首先会指导你填写一些项目信息：

name (your-project):
version (1.0.0):
description:
entry point (index.js):
git repository:
author:
license (MIT):

这些信息是可以不填写的，每一项都有默认值

package.json

执行上面的命令后，你会得到一个package.json内容如下：

{
  "name": "my-new-project",
  "version": "1.0.0",
  "description": "My New Project description.",
  "main": "index.js",
  "repository": {
    "url": "https://example.com/your-username/my-new-project",
    "type": "git"
  },
  "author": "Your Name <you@example.com>",
  "license": "MIT"
}

执行yarn init命令，实际就是创建这个package.json，没有其他工作了。这会让你感到简单和轻松，愉快的编辑package.json吧

package.json中包含你项目的信息：名字、维护人员名单、源代码地址，其中最最重要的是你项目所需要安装的依赖。

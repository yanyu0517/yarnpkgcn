yarn outdated

检查依赖是否过期。

yarn outdated

列出所有依赖的版本信息，包括当前安装版本，给予semver渴望的版本以及最新的版本。

比如，你的package.json是这样的：

"dependencies": {
  "lodash": "4.15.0",
  "underscore": "~1.6.0"
}

命令运行结果：

$ yarn outdated
Package    Current Wanted Latest
lodash     4.15.0  4.15.0 4.16.4
underscore 1.6.0   1.6.0  1.8.3 
✨  Done in 0.72s.
yarn outdated [package...]

列出指定依赖的版本信息，支持多个依赖查询。

例如，package.json与之前的例子相同，执行这个命令，你会看到将会输出你指定的依赖的信息：

$ yarn outdated lodash
Package Current Wanted Latest
lodash  4.15.0  4.15.0 4.16.4
✨  Done in 1.04s.

安装依赖

当你从代码仓库中下载了代码之后，需要安装项目所需要的依赖。

如果你运行yarn add命令，添加一个依赖，依赖的安装过程会自动完成。

安装依赖

yarn install命令用来安装一个项目的全部依赖。项目的这些依赖是在项目的package.json文件中描述的，并被储存在yarn.lock文件中

通常在下面两种情况下需要安装依赖

  1. 一个全新的项目刚刚被下载
  2. 依赖有了更新

安装命令参数

安装命令参数，包括：

  1. 安装所有依赖：yarn or yarn install
  2. 依赖只会安装一个版本，也就是flat：yarn install --flat
  3. 强制重新安装所有依赖yarn install --force
  4. 只安装production版本的依赖yarn install --production

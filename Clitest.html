yarn test

运行测试脚本。

yarn test

如果你在package.json文件中定义了scripts项，命令将会运行test脚本。

例如，如果你有一个bash脚本，放在scripts/test路径下：

#!/bin/bash

echo "Hello, world!"

package.json文件：

{
  "name": "my-tribute-package",
  "version": "1.0.0",
  "description": "This is not the best package in the world, this is just a tribute.",
  "main": "index.js",
  "author": "Yarn Contributor",
  "license": "MIT",
  "scripts": {
    "test": "scripts/test"
  }
}

运行yarn test：

$ yarn test
yarn test v0.15.1
$ "./scripts/test"
Hello, world!
✨ Done in 0.17s.
yarn run test

yarn test是yarn run test的缩写。

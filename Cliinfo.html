yarn info

显示包的信息。

yarn info <package> [<field>]

运行这个命令将会以树形结构展现包的信息。这个项目可以不是本地的，可以直接获得远程仓库中某一个包的信息。

$ yarn info react

yarn info vx.x.x
{ name: 'react',
  version: '15.4.0-rc.2',
  description: 'React is a JavaScript library for building user interfaces.',
  time: { modified: '2016-10-06T22:09:27.397Z', ... } ... }

默认的输出结构是用单引号序列化的，如果需要得到有效的JSON格式，可以使用--json参数：

yarn info react --json

{"type":"inspect","data":{"name":"react","time":{...}}}
{"type":"finished","data":417}

查看指定版本的信息

增加@[version]参数：

$ yarn info react@15.3.0

yarn info vx.x.x
{ name: 'react',
  version: '15.3.0',
  description: 'React is a JavaScript library for building user interfaces.',
  time: { modified: '2016-10-06T22:09:27.397Z', ... } ... }

选择信息项

如果增加了信息项参数（可选的），那么只有该部分的信息会返回。

$ yarn info react description

yarn info vx.x.x
React is a JavaScript library for building user interfaces.

如果指定的信息项是多层级的，那么它的子节点会被打印：

$ yarn info react time

yarn info vx.x.x
{ modified: '2016-10-06T22:09:27.397Z',
  created: '2011-10-26T17:46:21.942Z', ... }

yarn info react time --json
{"type":"inspect","data":{"modified":"2016-10-06T22:09:27.397Z","created":...}}
...

显示readme

因为readme 通常会非常长，yarn info默认是不打印readme 。如果需要，请添加readme参数：

$ yarn info react readme

yarn info vx.x.x
## react

An npm package to get you immediate access to
[React](https://facebook.github.io/react/).
...

贡献

无论贡献大小，我们都非常欢迎。在参与贡献之前，请阅读code of conduct.

构建 

yarn run build

yarn run watch

测试 

yarn run test

yarn run lint

发Pull Requests

我们积极欢迎您的pull requests.

  1. Fork Yarn的 仓库并从master创建您自己的分支.
  2. 如何您添加了应该测试的代码，请添加测试用例
  3. 如果您更改的API，请更新文档
  4. 确保测试套件全部通过
  5. 确保您的代码通过lints校验

授权

给Yarn做贡献，您同意将您的贡献按BSD license进行授权。

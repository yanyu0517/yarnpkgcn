管理依赖

新增、更新或者删除依赖命令。

每一个命令都会更新 package.json和yarn.lock文件。

新增依赖

当你需要使用其他依赖包，你首先新增依赖，运行下面的命令：

yarn add [package]

将会在package.json中自动添加[package]。同时也会更新yarn.lock中的相关依赖。

  {
    "name": "my-package",
    "dependencies": {
+     "package-1": "^1.0.0"
    }
  }

还可以新增其他类型的依赖 types of dependencies：

  • yarn add --dev to add to devDependencies
  • yarn add --peer to add to peerDependencies
  • yarn add --optional to add to optionalDependencies

你可以通过指定dependency version或者tag参数，来指定依赖包的版本

yarn add [package]@[version]
yarn add [package]@[tag]

[version]或者[tag]参数会在package.json文件中被明确声明出来。

例如：

yarn add package-1@1.2.3
yarn add package-2@^1.0.0
yarn add package-3@beta

{
  "dependencies": {
    "package-1": "1.2.3",
    "package-2": "^1.0.0",
    "package-3": "beta"
  }
}

更新依赖

yarn upgrade [package]
yarn upgrade [package]@[version]
yarn upgrade [package]@[dist-tag]

同时更新package.json和yarn.lock文件。

  {
    "name": "my-package",
    "dependencies": {
-     "package-1": "^1.0.0"
+     "package-1": "^2.0.0"
    }
  }

删除依赖

yarn remove [package]

同时更新package.json和yarn.lock文件

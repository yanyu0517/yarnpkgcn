yarn licenses

列出安装依赖的license.

yarn licenses ls

运行命令会得到所有通过yarn或者yarn install命令安装的依赖的license，以及源代码仓库地址，按字母顺序排序。 Running this command will list, in alphabetical order all of the packages that were installed by yarn or yarn install, and give you the license (and URL to the source code) associated with each package.

yarn licenses ls

yarn run

运行脚本。

yarn run [script]

如果你已经在package.json文件中定义了cripts，那么可以使用这个命令执行指定的[script]。

例如，

yarn run test

将会运行在package.json文件中定义的test脚本。

yarn run

如果没有为yarn run命令指定需要运行的脚本，那么将会列出所有能够执行的脚本。

package.json脚本

package.json文件中的scripts数据项中定义了可以执行的脚本。

例如，下面的例子定义了一个名为test的脚本，yarn run test将会执行jest框架，对代码进行自动化测试。

"scripts": {
  "test": "jest"
},

yarn global

安装全局依赖。

yarn global

yarn global可以作为若干命令的前缀使用，如add，bin，ls和remove。全局依赖与局部依赖的唯一不同就是，全局依赖存放的路径并不是项目目录中，而可以在操作系统中的任何目录。安装全局依赖的好处是，可以作为工具来使用。例子：create-react-app被安装在了全局：

$ yarn global add create-react-app
# the `create-react-app` command is now available globally:
$ create-react-app

可以开始加上yarn global的命令：

  • yarn add：安装依赖。
  • yarn bin：显示yarn的bin文件的位置。
  • yarn ls：列出所有已经安装的依赖
  • yarn remove删除依赖。
